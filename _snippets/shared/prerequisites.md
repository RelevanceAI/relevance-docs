## Prerequisites

Before building an integration, ensure you have:

### Required Permissions

<AccordionGroup>
  <Accordion title="API Access & Authentication">
    - Valid API credentials (API key, OAuth tokens, or other authentication method)
    - Appropriate scopes and permissions for the actions you want to perform
    - Access to the service's API documentation
  </Accordion>
  
  <Accordion title="Relevance AI Account">
    - An active Relevance AI account
    - Access to the Integrations & API Keys page
    - Permissions to create and configure tools
  </Accordion>
  
  <Accordion title="Service-Specific Requirements">
    Depending on the service you're integrating with, you may need:
    - Admin or developer access to the external service
    - Webhook configuration permissions (for trigger-based integrations)
    - Specific plan or subscription level that includes API access
  </Accordion>
</AccordionGroup>

### Understanding API Basics

<Info>If you're new to APIs, the `/integration` skill can help guide you through the process. You can also use the Invent feature to assist with API integration setup.</Info>

At minimum, you should understand:
- What an API endpoint is
- How to read API documentation
- Basic HTTP methods (GET, POST, PUT, DELETE)
- How authentication works for your target service
