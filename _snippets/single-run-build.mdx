Build is where different components are combined together to form the whole input to analysis steps.
![Build](/images/get-started/tool-build.png)

Once all is ready, provide all the required inputs and you can
1. execute each step in order by clicking on the "play" button on top right of the step component
2. Use the `Run all` button located on the top right of the page
![Single run - build](/images/get-started/tool-build-run.png)

Final analysis result will be shown at the bottom of the page. Noe that you can see the 
output of each step by clicking on `+ Expand` at the step.
