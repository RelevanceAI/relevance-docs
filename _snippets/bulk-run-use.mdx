On the "Use" tab, you have access to "Run in bulk" which is located next to "App".
![Bulk run - Tool](/images/general/tool-bulk-run.png)
Click on `Run on my data` button on the "Run in bulk" page.
This button directs you to the [Data](/get-started/key-concepts/data) page where you 
have access to "Enrich" (i.e. Bulk run on the data table) as explained above.
