1. Title: the title of your input component
2. Description: To describe what this input is or will be used for, etc.
3. Variable name: located on the bottom left and marked in green, you can rename the variable and use the name to access the 
data in your Tool
4. Optional/Required: located on the top right, as the name suggests, unlike an optional component, a required components 
must be provided before your analysis starts.
