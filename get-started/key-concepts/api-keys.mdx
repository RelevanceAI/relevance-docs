## What Are API Keys?

API keys are unique codes that let different software talk to each other. They act as a way to identify who is trying to connect and make sure the connection is secure.

Think of an API key like a special key card you use to enter a building. It tells the system who you are and what you're allowed to do. Every time an app wants to connect to another app, it shows this key card.

<Warning>
  You can think of API keys as a form of a password so these should never be shared.
</Warning>

Unlike custom API keys, native API key integrations connect to third-party services through pre-built connectors directly supported by Relevance AI. Add your API key once on the Integrations page, and Relevance AI automatically injects it wherever it's needed. No need to configure individual tool steps or reference keys with variables.

<Note>
  When you add an API key to the Integrations page, you leave the API key field blank in your tool steps. The system automatically detects and uses the key from your integrations.
</Note>

**Examples:** Firecrawl, Lusha, Airtop, OpenAI, Anthropic, Azure OpenAI, and other LLM providers
