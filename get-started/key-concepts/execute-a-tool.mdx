---

When we [design a Tool](/custom-tools/create-a-tool) or use an [existing template](/templates/introduction), 
we can 
1. [Single Run](/templates/single-run) the Tool: test the Tool/Template on individual sample inputs
2. [Bulk Run](/templates/bulk-run) the Tool: use a dataset as input and let the Tool/Template to fetch the input data directly from the source 
and apply the analysis to the whole dataset in one go.
