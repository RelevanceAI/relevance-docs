### Integrations

To get the most out of Super GTM, start by connecting your integrations. Click the gear icon next to the Super GTM toggle to open the settings panel, find the integration you want to enable, and click "Connect account" or "Add API key".

<img src="/images/super-gtm-settings.png" alt="Super GTM settings panel" />

There are two ways integrations can be set up:

- **OAuth:** Each user authorizes their own connection via a popup modal. Each user needs to connect their apps individually.
- **API key:** Add an API key to a Relevance project, which can be used by all users in that project. These can be centrally managed on a project basis in the builders app under [Integrations & API Keys](/get-started/core-concepts/api-integration).

Learn more about [OAuth authentication](/enterprise/user-level-authentication) and [API key integrations](/get-started/core-concepts/api-integration).

<Note>
Depending on how your organization has configured each external application, you may need to request admin access to enable OAuth access or to obtain an API key. Check with your IT team or admin of the external application to confirm.
</Note>

### Action approval

<Info>
The agent can never send, create, update, or delete anything without your explicit confirmation.
</Info>

For integrations that can modify data (e.g. sending an email, creating a CRM record, or posting a Slack message), Super GTM includes a built-in approval flow. Before the agent executes a write action, it will present the action and its parameters for your review. You can:

- **Approve** the action to let it proceed as-is.
- **Reject** the action to prevent it from executing.

This gives you human-in-the-loop control over any action that changes data in your connected tools.

## Using Super GTM

### Super Agent panel

The Super Agent floating panel gives you quick access to agent controls and scheduling directly from the chat interface. Click the panel icon to:

- **View agent details**: See which agent you're working with and its configuration
- **Manage schedules**: Create, edit, and delete [recurring schedules](/build/agents/build-your-agent/agent-triggers/scheduled-triggers#setting-up-a-recurring-schedule-from-the-super-agent-panel) for automated tasks
- **Access agent settings**: Quickly adjust agent behavior without leaving Chat

The panel is especially useful for setting up automation on the fly. For example, you can create a daily schedule to summarize new leads or send weekly reports, all without switching to the Agent Builder.

## Frequently asked questions

<AccordionGroup>
  <Accordion title="How do I get access to Super GTM?">
    Super GTM is in closed beta (started March 2, 2026) for Enterprise customers. Contact your Account Executive to discuss enrollment. Access is enabled on a per-user or per-domain basis by Relevance staff after discussion with your AE.
  </Accordion>

  <Accordion title="Can I disable the approval flow?">
    No. The approval flow is a core safety feature of Super GTM and cannot be disabled. It ensures you maintain control over all actions that modify data in your systems.
  </Accordion>

  <Accordion title="What happens if I reject an action?">
    If you reject an action, Super GTM will not execute it. The agent will ask how you'd like to proceed. You can provide different instructions, modify the action, or cancel the task entirely.
  </Accordion>

  <Accordion title="Can I use Super GTM with my existing agents?">
    Yes! Super GTM can run any agent you've built in Relevance AI. Just configure which agents to make available in the Super GTM settings.
  </Accordion>

  <Accordion title="What's the difference between OAuth and API key integrations?">
    OAuth creates individual connections where each user authorizes their own account via a popup modal. API keys are added to a Relevance project and shared across all users in that project. These can be centrally managed in the builders app at [app.relevanceai.com](https://app.relevanceai.com) under "Integrations & API Keys". Depending on your organization's configuration, you may need admin access for either method. Check with your IT team.
  </Accordion>

  <Accordion title="Does Super GTM work on mobile?">
    Yes! Super GTM is available on iOS via TestFlight. Install the [TestFlight app](https://apps.apple.com/us/app/testflight/id899247664), then [open the beta invite link](https://testflight.apple.com/join/xx1C1Z6y) to install Relevance AI. Turn on automatic updates in TestFlight to stay up-to-date.
  </Accordion>

  <Accordion title="What integrations are supported?">
    Currently available: HubSpot, GSuite (Calendar, Gmail, Docs), Linear, Notion, Slack, Cursor, Apollo, Salesforce, Gong, Microsoft Calendar, Microsoft Outlook, and Microsoft Teams. Google Drive, Google Sheets, Outreach, ZoomInfo, and Firmable are planned next. See the [Supported Integrations](#supported-integrations) table for full details on connection methods and permissions.
  </Accordion>

  <Accordion title="Can Super GTM access data from multiple tools at once?">
    Yes! Super GTM can work across multiple tools simultaneously. For example, it can pull data from your CRM, check your calendar, and draft an email, all in a single conversation.
  </Accordion>

  <Accordion title="Can I schedule recurring tasks from the Super Agent panel?">
    Yes! The Super Agent panel includes a Schedules tab where you can create, edit, and delete recurring schedules. This lets you set up automation directly from Chat without switching to the Agent Builder. Learn more about [panel-based scheduling](/build/agents/build-your-agent/agent-triggers/scheduled-triggers#setting-up-a-recurring-schedule-from-the-super-agent-panel).
  </Accordion>
</AccordionGroup>
