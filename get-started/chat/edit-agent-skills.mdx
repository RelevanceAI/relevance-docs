title: "Edit Agent Skills in Chat"
description: "Edit your agent's skills and prompt directly from Chat"
---

You can now edit agent skills directly from Chat using the floating panel. This lets you make quick adjustments to your agents while testing them, without switching back to the Agent Builder.

## Accessing the Agent Skills Panel

When you're chatting with an agent in Relevance Chat, you can access its skills through the floating panel:

1. Start a conversation with an agent by @ mentioning it
2. Look for the agent skills panel in the Chat interface
3. Click on any skill to open the editor

## Editing Agent Skills

The Chat editor provides the same visual editing experience as the main Agent Builder:

### Visual Pill Editor

When you edit an agent's system prompt in Chat, tool and action references appear as visual pills with icons and readable names. This makes it easy to:

- See which tools your agent is using at a glance
- Identify tools by their icons and names
- Keep your prompt organized and readable

### Using the Slash Menu

Type **/** in the prompt editor to:

- Reference existing tools
- Add tool placeholders
- Insert markdown formatting

Tool references automatically appear as visual pills when you select them from the slash menu.

## Benefits of Editing in Chat

Editing agent skills directly in Chat enables faster iteration:

- **Quick adjustments** - Make changes without leaving your testing environment
- **Immediate testing** - See how prompt changes affect agent behavior right away
- **Context-aware editing** - Edit while you have the conversation context fresh in mind
- **Same visual editor** - Uses the same pill-based editor as the Agent Builder for consistency

## What You Can Edit

From the Chat floating panel, you can edit:

- System prompt and instructions
- Tool references in the prompt
- Agent configuration settings

<Note>
Changes made in Chat are saved to the agent and will apply to all future conversations with that agent.
</Note>

## Editing vs. Building

While you can edit agent skills in Chat, some agent configuration tasks are still best done in the main Agent Builder:

- Creating new tools from scratch
- Configuring complex tool settings
- Setting up triggers and escalations
- Managing knowledge bases

For these tasks, navigate to the Agent Builder from the main Relevance AI platform.
