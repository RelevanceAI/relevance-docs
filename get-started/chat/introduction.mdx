![Relevance Chat](/images/hero-chat.png)

Chat is where you interact with your AI Agents and Workforces. @ mention any Agent, combine multiple Agents in one conversation, or use pre-built Agents for tasks like building websites, generating images, and deep research — all from [chat.relevanceai.com](https://chat.relevanceai.com), on desktop and mobile.

Once you build an Agent or clone one from the Marketplace, you can use it in Chat — and [control which Agents are visible](/get-started/chat/using-agents#agent-visibility) to your team.

## Getting started

Navigate to [chat.relevanceai.com](https://chat.relevanceai.com) to start chatting. Chat works on desktop and mobile browsers, and is also available as a desktop app.

### Selecting an LLM

You can choose which LLM to use for your conversation, or select **Auto** to let Chat pick the best model for your request.

## Recent meeting chat links (desktop app)

When no conversation is active, the desktop app displays up to 5 recent meeting chat links on the home view. These links appear after meetings end, giving you quick access to the chat conversation from each meeting. Each link shows a platform-specific icon, the meeting title, and a timestamp. Click any link to open that meeting's chat directly.

<Note>
Recent meeting chat links are only available in the desktop app. They do not appear in the web version of Chat.
</Note>
