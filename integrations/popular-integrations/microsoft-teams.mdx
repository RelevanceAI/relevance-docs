# Microsoft Teams

Microsoft Teams is a collaboration platform that combines workplace chat, video meetings, file storage, and application integration.

The Microsoft Teams integration in Relevance AI was built by Relevance AI, and is therefore supported by our team, not Microsoft. If you have a question or issue with using Microsoft Teams in Relevance AI, please reach out to our support team. If you have a question or issue that is only about Microsoft Teams, you can reach out to Microsoft support.

​
## Connect the integration

Connecting your Microsoft account to Relevance AI is a straightforward process. The Microsoft integration provides unified authentication across Teams, Outlook, SharePoint, and OneDrive.

1. Go to the "Integrations & API Keys" page in the sidebar of your Relevance AI dashboard.
2. Click on "Microsoft (Teams, Outlook, SharePoint, OneDrive)" from the available integrations.
3. Click on the "Add Integration" button.
4. Sign in with your Microsoft account.
5. Approve the requested permissions.
6. Once connected, you'll see your Microsoft account listed under connected integrations.

Note: You may need admin consent depending on your organisation's Microsoft policies.

​
## Microsoft integration overview

Relevance AI uses a unified Microsoft authentication that works across multiple Microsoft services:

- **Microsoft Teams** - Monitor and respond to Teams messages
- **Outlook** - Access email and calendar
- **SharePoint** - Connect to SharePoint sites and documents
- **OneDrive** - Access files stored in OneDrive

This means connecting one Microsoft account gives you access to all these services within Relevance AI.

​
## Setting up triggers

Microsoft Teams triggers activate when new messages are posted in channels or chats. When a trigger fires, the message content is passed into your Agent or Workforce as input.

**Important**: Teams triggers activate only on new messages. They do not trigger on new chat creation, group creation, or webhooks.

To set up Microsoft Teams as a trigger:

1. Create or edit an agent in Relevance AI.
2. Navigate to the "Triggers" section in "Agent Profile".
3. Click "Add Trigger".
4. Select "Microsoft Teams" as the trigger source.
5. Choose your connected Microsoft account.
6. Select the Team and Channel (or chat) to monitor.
7. Configure any trigger conditions if applicable.
8. Save your trigger configuration.

Your Agent will now run automatically whenever a new message is posted in the selected location.

## Tool steps for Microsoft Teams

The Microsoft Teams integration provides actions that your agents can use to send messages and manage Teams. These actions can be incorporated into your agent's workflows as tool steps.

**Note**: Most Microsoft Teams tool steps are currently in beta. Please report any issues to our support team.

​
### Messaging

**Send Channel Message**
Post a message to a specific Teams channel.

**Send Chat Message**
Send a direct message in Teams.

​
### Channel Management

**Create Channel**
Create a new channel in a Team.

**List Channels**
Get all channels in a Team.

​
### Other Actions

**List Shifts**
Retrieve shift information from Teams.

Type "Microsoft Teams" in the tool step search bar to see all available Microsoft Teams actions when building your tools.

​
## Use the integration's API tool step (Advanced)

In addition to the pre-built actions available in the tool directory, you can build custom tools that perform Microsoft Teams-specific activities using the Microsoft Teams API Call tool step.

You can build custom tools that perform Microsoft Teams activities, by using the Microsoft Teams API Call tool-step:

- Create a new tool.
- Scroll down to Tool-steps.
- Add Microsoft Teams API tool-step.
- Select your connected Microsoft account in the dropdown.
- Configure the API endpoint and parameters according to the Microsoft Graph API documentation.
- Test your API call to ensure it works as expected.

The Microsoft Teams API Call tool step uses the Microsoft Graph API, which provides access to the full range of Microsoft Teams functionality.

​
## Remove integration

If you need to remove the integration for Microsoft Teams:

1. Go to the Integrations & API Keys page from the sidebar.
2. Search for Microsoft (Teams, Outlook, SharePoint, OneDrive) from the list.
3. Click "…" on the account you want to remove.
4. Click "Remove" and confirm your choice.

Note: Removing the Microsoft integration will disable all triggers and tool steps using this account across Teams, Outlook, SharePoint, and OneDrive.

## Frequently asked questions (FAQs)

**What is Microsoft Teams and why should I use it?**

Microsoft Teams is a collaboration platform that combines workplace chat, video meetings, and file storage. Using it with Relevance AI allows you to automate responses to messages, trigger workflows from Teams conversations, and send automated messages to channels or chats.

**Do I need a Microsoft Teams account to use this integration?**

Yes, you need a Microsoft account with access to Microsoft Teams. This integration is designed for Microsoft Teams for business and enterprise.

**Can I use multiple Microsoft accounts with Relevance AI?**

Yes, you can connect multiple Microsoft accounts to your workspace. Each account can be used for different triggers and tool steps.

**What permissions does the Microsoft Teams integration require?**

Relevance AI requires permissions to read messages, send messages, and access team/channel information. The exact permissions will be shown during the OAuth flow when you connect your account.

**Do triggers work for private messages?**

Yes, Teams triggers can monitor both channel messages and private chats, depending on your configuration.

**Can I filter which messages trigger my Agent?**

Yes, you can configure trigger conditions such as keyword matching to filter which messages activate your workflow.

**How do I remove the Microsoft Teams integration?**

Follow the steps in the "Remove integration" section above. Note that removing the Microsoft integration affects all Microsoft services (Teams, Outlook, SharePoint, OneDrive).
