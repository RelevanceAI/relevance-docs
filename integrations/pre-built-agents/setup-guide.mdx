# Setup guide

Get your pre-built agent up and running in two simple steps.

<Steps>
  <Step title="Find your agent">
    Clone the agent from the Marketplace
  </Step>

  <Step title="Configure credentials">
    Add your API key and settings (some agents use direct integrations and don't need API keys)
  </Step>
</Steps>

---

## Step 1: Find your agent

1. Navigate to the [**Marketplace**](https://app.relevanceai.com/marketplace/f1db6c/0fff163c-daae-4d17-b878-0b4b4d4d902c) in Relevance AI
2. Search for "agent" in the search bar to view all available pre-built agents
3. Look for the platform name in the title (e.g., "Webflow Agent", "Notion Agent")
4. Click the **Clone** button

## Step 2: Configure your agent

<Warning>
Complete this setup before first use. For agents with direct integrations (Notion, Gmail, etc.), you'll connect your account via OAuth. For API-key agents (Webflow), you must add your API credentials before using your agent, otherwise you'll see authentication errors.
</Warning>

Choose your setup path based on which agent you cloned:

<Tabs>
  <Tab title="Direct integrations">
    **For example:** Notion, HubSpot, Gmail, Google Docs, Google Slides

    <div style={{ width:"100%",position:"relative",paddingTop:"56.25%"}}>
    <iframe src="https://app.supademo.com/embed/cmhlkrbw80gg5la10v1na0qm7" frameBorder="0" title="Connect direct integration agent" allow="clipboard-write; fullscreen" webkitAllowFullscreen="true" mozAllowFullscreen="true" allowFullscreen style={{ position:"absolute",top:0,left:0,width:"100%",height:"100%",border:"3px solid #5E43CE",borderRadius:"10px" }} />
    </div>

    **Setup steps:**

    1. Go to the **Tools** tab in your agent
    2. Click **Add account** on the integration tool
    3. Sign in with your account (e.g., Google, Notion, HubSpot)
    4. Grant the requested permissions

    Your agent is ready to use.
  </Tab>

  <Tab title="API key setup">
    **For example:** Webflow

    <div style={{ width:"100%",position:"relative",paddingTop:"56.25%"}}>
    <iframe src="https://app.supademo.com/embed/cmhmn3zp7094edqxay6trr9uj" frameBorder="0" title="Configure API key agent" allow="clipboard-write; fullscreen" webkitAllowFullscreen="true" mozAllowFullscreen="true" allowFullscreen style={{ position:"absolute",top:0,left:0,width:"100%",height:"100%",border:"3px solid #5E43CE",borderRadius:"10px" }} />
    </div>

    **Setup steps:**

    1. Get your API key from the platform (see accordion below for help)
    2. After cloning, a setup popup should appear → Fill in the values
    3. If the popup doesn't appear:
       - Go to the **Variables** tab at the top of your agent
       - Fill in the required fields (API Key, Site ID, etc.)
       - Leave the **Documentation** field unchanged (it's pre-loaded)
       - Click **Save**

    <AccordionGroup>
      <Accordion title="What is an API key?" icon="key">
        Think of an API key as a secure password that lets the agent access your account on your behalf. It's like giving the agent a special key to your workspace so it can create, update, and manage things for you—without needing your actual login password.

        **Why you need it:**
        - The agent needs permission to make changes in your account
        - API keys are safer than sharing your password
        - You can revoke the key anytime if needed
      </Accordion>

      <Accordion title="How to find your API key" icon="magnifying-glass">
        API keys are managed by the external platform provider (Webflow, etc.), not within Relevance AI. You'll need to navigate to your account on the respective platform to generate or locate your API key.

        Each platform has its own API documentation and guides. For additional help, search for "[Platform Name] API key" (e.g., "Webflow API key") to find official setup guides and support resources.
      </Accordion>

      <Accordion title="Configuration checklist for Webflow" icon="list-check">
        For the Webflow agent, you'll need to provide:

        - **API Key** - Your Webflow API token
        - **Site ID** - Your specific Webflow site identifier

        → [How to get your Webflow Site ID](/integrations/pre-built-agents/webflow-agent#how-to-get-your-webflow-site-id)
      </Accordion>
    </AccordionGroup>
  </Tab>
</Tabs>
