### Built-in safety

The agent includes smart guardrails:
- **Read operations** (searches, lookups) happen instantly
- **Write operations** (create, update, delete) require your confirmation first
- **Bulk changes** (>100 records) require double-confirmation

## HubSpot agent limitations

The HubSpot agent is designed specifically for **core CRM database operations** and includes specialized tools to handle HubSpot's specific requirements:

- **Date range formatting** - Uses a dedicated tool to convert dates to Unix timestamp format required by HubSpot's API
- **Math calculations** - Uses a specialized math tool for accurate numerical operations like lead scoring, revenue calculations, and date arithmetic

The agent focuses on managing contacts, companies, deals, tickets, engagements, properties, and associations. Platform features like workflows, marketing emails, sales sequences, and CMS are outside its scope.
