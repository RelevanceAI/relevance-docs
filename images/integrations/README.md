# Integration Images

This directory contains screenshots and images for integration documentation pages.

## Microsoft Teams Integration Images (Required)

The following images are referenced in the Microsoft Teams integration documentation but need to be added:

1. **microsoft-teams-integration.png**
   - Screenshot of the Integrations & API Keys page showing the Microsoft Teams integration
   - Should show the "Microsoft (Teams, Outlook, SharePoint, OneDrive)" integration option

2. **microsoft-teams-trigger-selection.png**
   - Screenshot of the trigger selection UI
   - Should show the Microsoft Teams option being selected as a trigger source

3. **microsoft-teams-trigger-config.png**
   - Screenshot of the trigger configuration page
   - Should show the Team and Channel selection dropdowns and trigger setup options

## Adding Images

When adding images to this directory:
- Use descriptive, kebab-case filenames
- Optimize images for web (compress if needed)
- Ensure screenshots are clear and readable
- Include relevant UI elements without sensitive information
