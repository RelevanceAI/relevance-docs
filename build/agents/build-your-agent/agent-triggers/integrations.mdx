### External Triggers

External triggers like **Zapier** are configured and managed outside of Relevance AI. These allow you to connect your Agent to third-party automation platforms without any setup within Relevance AI itself.

## Pausing integration triggers

You can pause any integration trigger without disconnecting the integration or losing your configuration. This is useful when you need to temporarily stop your agent from responding to events while keeping everything set up for when you're ready to resume.

**To pause an integration trigger:**

1. Go to your agent's **Triggers** section in agent settings
2. Find the integration trigger you want to pause (Gmail, Slack, LinkedIn, WhatsApp, etc.)
3. Click the pause button next to the trigger

**To resume:**

Click the resume button next to the paused trigger. Your agent will start responding to events again immediately.

### What happens when you pause an integration trigger

- The trigger stops processing new events from the integration
- Your integration connection remains active
- All trigger configuration is preserved
- You can resume at any time without reconfiguring

### Common use cases

- **Testing changes**: Pause production triggers while you test updates to your agent
- **Maintenance windows**: Temporarily disable triggers during system updates
- **Controlling volume**: Pause high-traffic triggers during busy periods
- **Debugging**: Isolate specific integration triggers to troubleshoot issues

<Note>
Pausing an integration trigger does not disconnect your integration. Your credentials and connection remain active, and you can resume the trigger without re-authenticating.
</Note>

## Frequently asked questions (FAQs)

<AccordionGroup>
  <Accordion title="How do I connect an integration trigger?">
    Navigate to your agent's Triggers section, select the integration you want to connect, and authenticate with your credentials for that service.
  </Accordion>
  <Accordion title="Are premium triggers included in my plan?">
    Premium triggers (LinkedIn, WhatsApp, WhatsApp for Business, and Telegram) are available on the Pro plan and above. Check the [pricing page](/admin/subscriptions/new-pricing) for details.
  </Accordion>
  <Accordion title="Can I pause an integration trigger without disconnecting it?">
    Yes. Pausing an integration trigger stops it from processing events while keeping your integration connected and all configuration intact. You can resume the trigger at any time without re-authenticating or reconfiguring.
  </Accordion>
  <Accordion title="What's the difference between pausing a trigger and disconnecting an integration?">
    Pausing a trigger temporarily stops it from firing but keeps everything configured and ready to resume. Disconnecting an integration removes the connection entirely and requires re-authentication to set up again. Use pause when you need temporary control; disconnect only when you're done with the integration.
  </Accordion>
</AccordionGroup>
