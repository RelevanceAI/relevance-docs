The Performance dashboard provides detailed, interactive analysis of your Agent evaluation results. Located in the **Monitor** tab under **Performance**, this dashboard helps you explore eval runs, understand pass/fail patterns, and identify areas for improvement through visual breakdowns and conversation-level insights.

## Getting Started

<Steps>
  <Step title="Access Performance">
    Navigate to the **Monitor** tab in your Agent builder and select **Performance** from the left sidebar
  </Step>
  <Step title="View Your Eval Runs">
    The dashboard displays a paginated table of all your evaluation runs with key metrics
  </Step>
  <Step title="Explore Results">
    Click on any row to expand and see detailed per-evaluator breakdowns, or click chart bars to filter results
  </Step>
</Steps>

## Dashboard Components

### Eval Runs Table

The main table displays your evaluation runs with the following information:

| Column | Description |
|--------|-------------|
| **Run Name** | Name of the evaluation run with expandable arrow |
| **Pass/Fail Chart** | Visual bar chart showing pass (green) and fail (red) proportions |
| **Score** | Overall pass percentage (e.g., "75%") |
| **Date** | When the evaluation was executed |
| **Actions** | "View Conversation" button to see full details |

<Tip>
The table is paginated—use the controls at the bottom to navigate through your evaluation history. You can adjust the number of rows displayed per page.
</Tip>

### Interactive Chart Filtering

Each eval run displays a horizontal bar chart showing the proportion of passed (green) and failed (red) evaluations:

- **Click on the green section** to filter the table and show only runs with passing evaluations
- **Click on the red section** to filter the table and show only runs with failing evaluations
- **Click again** to remove the filter and return to the full view

This allows you to quickly focus on successful runs or identify problematic evaluations that need attention.

### Expandable Row Details

Click the arrow next to any run name to expand the row and see a detailed breakdown:

<AccordionGroup>
  <Accordion title="Per-Evaluator Breakdown">
    Each evaluator (Judge) is listed with:
    - **Evaluator name** (e.g., "Professional Tone", "Empathy Shown")
    - **Pass/fail count** (e.g., "3 passed, 1 failed")
    - **Detailed reasons** explaining why evaluations passed or failed
    
    This breakdown helps you understand which specific criteria your Agent is meeting or missing.
  </Accordion>
  
  <Accordion title="Pass/Fail Reasons">
    For each evaluator, you'll see:
    - **Pass reasons**: Specific examples of what the Agent did well
    - **Fail reasons**: Detailed explanations of what went wrong and why the evaluation failed
    
    These insights help you identify patterns and make targeted improvements to your Agent's behavior.
  </Accordion>
</AccordionGroup>

### Conversation Viewer

Click the **View Conversation** button on any eval run to open a sidebar that displays:

1. **Full conversation transcript** between the simulated user and your Agent
2. **All Judge verdicts** with detailed pass/fail explanations
3. **Scenario context** showing the test scenario that was executed

This gives you complete visibility into what happened during the evaluation and why Judges reached their verdicts.

## Filter and Navigation Controls

The Performance dashboard includes several controls to help you navigate and filter your data:

### Filter Dropdowns

Use the filter dropdowns at the top of the table to:

- **Filter by pass/fail status** - Show only passing runs, failing runs, or all runs
- **Filter by date range** - Focus on evaluations from specific time periods
- **Filter by evaluator** - See runs that included specific Judges

### Pagination Controls

At the bottom of the table, you'll find:

- **Rows per page selector** - Choose how many runs to display (e.g., 10, 25, 50)
- **Page navigation** - Move between pages with previous/next buttons
- **Page indicator** - Shows current page and total pages (e.g., "Page 1 of 5")

## Relationship to Evals

The Performance dashboard complements the [Evals](/build/agents/build-your-agent/evals) section:

- **Evals section** - Create Tests, define Judges, and run evaluations
- **Performance dashboard** - Analyze and explore the results of those evaluations

While the Runs section in Evals shows basic evaluation history, the Performance dashboard provides:

- **Interactive exploration** with expandable rows and chart filtering
- **Detailed per-evaluator breakdowns** showing exactly which Judges passed or failed
- **Visual representations** of pass/fail proportions for quick pattern recognition
- **Conversation-level insights** with full transcripts and Judge reasoning

<Note>
For detailed performance analysis and interactive exploration of your eval runs, use the Performance dashboard. For creating and managing Tests and Judges, use the [Evals](/build/agents/build-your-agent/evals) section.
</Note>

## Frequently Asked Questions (FAQs)

<AccordionGroup>
  <Accordion title="How is the score calculated?">
    The score is the percentage of evaluations that passed. For example, if 3 out of 4 Judges passed, the score would be 75%.
  </Accordion>
  
  <Accordion title="What happens when I click on a chart bar?">
    Clicking the green section filters the table to show only runs with passing evaluations. Clicking the red section shows only failing runs. Click again to remove the filter.
  </Accordion>
  
  <Accordion title="Can I export Performance data?">
    Currently, the Performance dashboard is designed for interactive exploration within the platform. For detailed analysis, use the expandable rows and conversation viewer to review specific runs.
  </Accordion>
  
  <Accordion title="How far back does the Performance dashboard show data?">
    The Performance dashboard displays all evaluation runs for your Agent. Use the pagination controls to navigate through your complete evaluation history.
  </Accordion>
  
  <Accordion title="I don't see the Performance dashboard. How do I get access?">
    The Performance dashboard is being rolled out progressively, starting with Enterprise customers. If you're an Enterprise customer and don't see Performance in the Monitor tab yet, reach out to your account manager to discuss access.
  </Accordion>
</AccordionGroup>
