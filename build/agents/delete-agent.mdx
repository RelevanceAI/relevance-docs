You can delete agents from the Relevance AI interface or from within Inventor. Deleted agents can be restored if needed.

## Delete via the UI

1. Click **Agents** in the sidebar
2. Click the three-dot menu next to the agent you want to delete, then select **Delete**
3. Confirm the deletion in the dialog that appears

## Delete via Inventor

You can ask Inventor to delete an agent during a conversation.

1. Open Inventor from the sidebar
2. Ask Inventor to delete an agent, for example: "Delete this agent" or "Delete the [agent name] agent"
3. Inventor will prompt you to confirm before proceeding — confirmation is always required for deletions, even when auto-approve is enabled for other actions

<Note>
If you delete the agent you are currently editing in Inventor, you will be automatically redirected away from that agent's view.
</Note>

## Restore a deleted agent

Deleted agents can be restored via Inventor.

1. Open Inventor from the sidebar
2. Ask Inventor to restore the agent, for example: "Restore the [agent name] agent"
3. Inventor will prompt you to confirm the restoration — confirmation is always required for restore actions, even when auto-approve is enabled

## Frequently asked questions (FAQs)

<AccordionGroup>
    <Accordion title="What happens if my deleted agent is clonable?">If your deleted agent is clonable, it will no longer be available for cloning. People who have already cloned it will still be able to use their copy.</Accordion>
    <Accordion title="I accidentally deleted an agent, can I recover it?">Yes, deleted agents can be restored via Inventor. Ask Inventor to restore the agent by name and confirm when prompted.</Accordion>
