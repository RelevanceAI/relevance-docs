These Tool steps let you call any Airtop API endpoint that doesn't have a dedicated Tool step. Two variants are available depending on how you access Airtop:

- **Airtop - API call** â€” for customers who have connected their own Airtop API key (BYOK).
- **AI Browser - API call** â€” for customers using Relevance AI's managed AI Browser, with no personal Airtop account required. Requires the Starter plan or higher.

Use these steps for Airtop endpoints not yet covered by dedicated Tool steps (such as hover, scrape-content, form-filler, or file-input), or to call a beta endpoint before native support is added.

## Add the 'Airtop - API call' Tool step

You can add the 'Airtop - API call' Tool step to your Tool by:

1. Creating a new Tool, then clicking `+ Add Step`
2. Searching for and adding the 'Airtop - API call' Tool step
3. Clicking 'Expand' to see the full Tool step
4. **Method** (Required): Select the HTTP method for the request (`GET`, `POST`, `PUT`, `PATCH`, or `DELETE`).
5. **Relative path** (Required): Enter the relative path on `api.airtop.ai` (e.g., `/api/v1/sessions/{session_id}/windows/{window_id}/page-query`). Type `{{` to select a variable from a previous step.
6. **Body** (Optional): Enter a JSON object or array to send as the request body. Type `{{` to select a variable or enter the JSON directly.
7. **Headers** (Optional): Add custom request headers as key-value pairs. The Airtop authentication header is set automatically and cannot be overridden.
8. **URL Query Parameters** (Optional): Add query parameters to append to the request URL as key-value pairs.
9. Clicking `Run step` to test out the Tool

The step sends the request using your connected Airtop API key and returns `response_body`, `status`, `response_headers`, and `airtop_credits_used`. `airtop_credits_used` is informational only â€” BYOK calls bill against your own Airtop subscription, not against Relevance credits.

## Add the 'AI Browser - API call' Tool step

<Note>
  This step uses Relevance AI's managed Airtop key. You do not need to connect a personal Airtop account. Requires the Starter plan or higher.
</Note>

You can add the 'AI Browser - API call' Tool step to your Tool by:

1. Creating a new Tool, then clicking `+ Add Step`
2. Searching for and adding the 'AI Browser - API call' Tool step
3. Clicking 'Expand' to see the full Tool step
4. **Method** (Required): Select the HTTP method for the request (`GET`, `POST`, `PUT`, `PATCH`, or `DELETE`).
5. **Relative path** (Required): Enter the relative path on `api.airtop.ai` (e.g., `/api/v1/sessions/{session_id}/windows/{window_id}/page-query`). Type `{{` to select a variable from a previous step.
6. **Body** (Optional): Enter a JSON object or array to send as the request body.
7. **Headers** (Optional): Add custom request headers as key-value pairs.
8. **URL Query Parameters** (Optional): Add query parameters to append to the request URL.
9. Clicking `Run step` to test out the Tool

The step sends the request through Relevance AI's managed Airtop connection and returns `response_body`, `status`, `response_headers`, and `credits_used`. `credits_used` is charged to your Relevance credit balance â€” LLM-driven endpoints pass through Airtop's reported cost, while endpoints that don't return a credit count bill a flat fallback charge.

## Common use cases

- Calling Airtop endpoints not yet covered by dedicated Tool steps (hover, scrape-content, form-filler, file-input)
- Calling beta Airtop endpoints before native support is added
- Passing window or session IDs from previous steps into endpoints that require them

## Frequently asked questions (FAQs)

<AccordionGroup>
  <Accordion title="Which step should I use â€” 'Airtop - API call' or 'AI Browser - API call'?">
    Use 'Airtop - API call' if you have connected your own Airtop API key (BYOK) in the Airtop integration settings. Use 'AI Browser - API call' if you are using Relevance AI's managed AI Browser and have not connected a personal Airtop account.
  </Accordion>
  <Accordion title="Where can I find available Airtop endpoints?">
    Refer to the [Airtop API documentation](https://docs.airtop.ai/) for the full list of endpoints and their request and response schemas.
  </Accordion>
  <Accordion title="How do I pass session or window IDs from earlier steps?">
    Type `{{` in any field to select a variable from a previous step's output. For example, you can reference the session ID returned by 'Airtop - Create Window' when calling endpoints that require a session identifier in the path or body.
  </Accordion>
  <Accordion title="How are credits billed?">
    For 'Airtop - API call' (BYOK), Relevance AI does not charge credits for the call itself â€” you are billed directly by Airtop, and `airtop_credits_used` in the output is informational. For 'AI Browser - API call' (managed), `credits_used` is charged to your Relevance credit balance.
  </Accordion>
</AccordionGroup>
