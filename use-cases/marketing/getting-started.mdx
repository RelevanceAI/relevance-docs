Marketing teams are stretched across channels. AI Agents take the high-volume execution work — drafting nurture sequences, repurposing content, pulling cross-channel reports — so marketers can spend their time on strategy and storytelling.

## Start with a Marketplace template

The Marketplace ships pre-built Agents you can clone in one click. Adapt them to your brand voice, your channels, your funnel.

  ## Connect the tools it needs

  Marketing Agents work best when they can read from and write to the systems your team already uses.

  - **Marketing automation** — [HubSpot](/integrations/popular-integrations/hubspot) or Marketo for campaigns and lifecycle
  - **Email** — [Gmail](/integrations/popular-integrations/gmail) or Outlook for outbound
  - **Content & social** — [Notion](/integrations/popular-integrations/notion), [Canva](/integrations/popular-integrations/canva), [Slack](/integrations/popular-integrations/slack) for content workflows
  - **Analytics** — [Google Sheets](/integrations/popular-integrations/google-sheets) or your BI tool for performance pulls

  Full list at [Integrations](/integrations/introduction).

  ## Drop into code when you need to

  Programmatic GTM covers the building. Two escape hatches for when canned Tool steps fall short:

  - **Custom Python and JS code blocks** live inside Tools. Parse a weird analytics export, apply business logic to segmentation, transform campaign data — drop a code block in and write the logic directly.
  - **The JS SDK** lets you call Agents and Tools from your own Node.js app. Useful when embedding into a custom dashboard or wiring up a marketing webhook handler.

## The four levels in marketing

How the [autonomy ladder](/use-cases/guides) plays out across your funnel:

<CardGroup cols={2}>
  <Card title="L1 — Assisted" icon="hand">
    A marketer asks the Agent for three subject-line options for tomorrow's drop — the Agent returns options with reasoning.
  </Card>
  <Card title="L2 — Copilot" icon="map" href="/build/agents/give-your-agent-tasks/approval-mode">
    A marketer asks for a 5-step onboarding sequence for new free-trial users — the Agent drafts all 5, the marketer reviews and queues.
  </Card>
  <Card title="L3 — Autopilot" icon="bolt" href="/build/workforces/create-a-workforce">
    A new MQL lands in HubSpot — segmentation, nurture-sequence drafting, and Slack alerts to the AE all happen automatically.
  </Card>
  <Card title="L4 — Self-driving" icon="infinity">
    "Increase trial-to-paid conversion by 15%" — the Agent A/B tests onboarding sequences, send times, and CTAs against conversion, evolves its playbook.
  </Card>
