The 'Confluence - Create Post' Tool step allows you to add a new blog post to one of your Confluence spaces.

<Note>You can learn more about the Confluence integration [here](/integrations/popular-integrations/confluence).</Note>

You can add the 'Confluence - Create Post' Tool step to your Tool by:

1. Creating a new Tool, then searching for the 'Confluence - Create Post' Tool step
2. Click 'Expand' to see the full Tool step
3. Select or add your Confluence account
4. Select the Confluence space you want to create the blog post in under 'Space ID'
5. Add the title of your blog post
6. Add the content you want to add to the blog post under 'Body'
7. Use the 'Status' field to choose if you want to create the blog post published ('current') or as a draft.
8. Click 'Run Step' to test out your Tool step
