You can add the 'Trello - Add Checklist' Tool step to your Tool by:

1. Creating a new Tool, then searching for the 'Trello - Add Checklist' Tool step
2. Click 'Expand' to see the full Tool step
3. Select or add your Trello account
4. Select the board your card is on
5. Select the card you want to add a checklist to
6. Name the checklist in the 'Checklist Name' field
7. If you wish to copy the items from another checklist, add this checklist's name to 'Copy from Checklist'
8. Use the 'Position' field to choose whether the checklist is added above or below other checklists on the card
9. Click 'Run step' to test out the Tool step! 

<Info>You will not be able to add checklist items to the checklist item you create in this Tool step unless you copy items from another checklist using the 'Copy from Checklist' field. To create new items, you should add another Tool / Tool step using the 'Trello - Create a Checklist Item' Tool step.</Info>
