You can add the ‘List Labels’ Tool step to your Tool by:

1. Creating a new Tool, then searching for the ‘List Labels’ Tool step.
2. Click ‘Expand’ to see the full Tool step.
3. Select the account you would like to use.
4. Click 'Run step' to test out the Tool.
