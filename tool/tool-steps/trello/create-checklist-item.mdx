The 'Trello - Create a Checklist Item' Tool step allows you to add a checklist item to one of your Trello checklists. 

<Note>You can learn more about the Trello integration [here](/integrations/popular-integrations/trello).</Note>

## Add the 'Trello - Create a Checklist Item' Tool step to your Tool

<Note>In order to create a checklist item, you need to create a checklist first. You can do this in Trello or by using the [Add Checklist Tool step](/tool/tool-steps/trello/add-checklist).</Note>

You can add the 'Create a Checklist Item' Tool step to your Tool by:

1. Creating a new Tool, then searching for the 'Create a Checklist Item' Tool step
2. Click 'Expand' to see the full Tool step
3. Select or add your Trello account
4. Add the board your checklist is on
5. Add the card your checklist is on
6. Select the name of the checklist you want to add a checklist item to
7. Add the name of the checklist item in the 'Name' field
8. Use the 'Position' dropdown to specify whether the checklist item is added to the top or bottom of the checklist
9. Use the 'Checked' checkbox to specify whether the checklist item should be created as checked or not
10. Click 'Run step' to test your Tool step!
