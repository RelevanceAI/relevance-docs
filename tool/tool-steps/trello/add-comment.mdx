The 'Trello - Add Comment' Tool step allows you to add a comment to one of your Trello cards. 

<Note>You can learn more about the Trello integration [here](/integrations/popular-integrations/trello).</Note>

You can add the 'Trello - Add Comment' Tool step to your Tool by:

1. Creating a new Tool, then searching for the 'Trello - Add Comment' Tool step
2. Click 'Expand' to see the full Tool step
3. Select or add your Trello account
4. Select the board your card is on
5. Select the card you want to add a comment to
6. Add the text that you want to add in a comment to 'Comment'
7. Click 'Run step' to run your Tool step
