The 'Trello - Archive Card' Tool step allows you to archive a card in Trello from Relevance AI.

<Note>You can learn more about the Trello integration [here](/integrations/popular-integrations/trello).</Note>

You can add the 'Trello - Archive Card' Tool step to your Tool by:

1. Creating a new Tool, then searching for the 'Trello - Add Comment' Tool step
2. Click 'Expand' to see the full Tool step
3. Select or add your Trello account
4. Select the board your card is on
5. Select the card you want to archive
6. Click 'Run step' to run your Tool step
