The Website Builder Agent is a pre-built Agent that can build websites for you in Chat. You can use it independently in Single-agent mode, or with your other Agents, as well as other Chat Agents, in Multi-agent mode.

<AccordionGroup>
    <Accordion title="How do I use the Website Builder Agent?">You can use the Website Builder Agent by @ mentioning it in Chat.</Accordion>
    <Accordion title="Can I use the Website Builder Agent with my other Agents?">Yes, you can use the Website Builder Agent with your other Agents.</Accordion> 
    <Accordion title="Can I use the Website Builder Agent with other Chat Agents?">Yes, you can use the Website Builder Agent with other Chat Agents.</Accordion>
    <Accordion title="How many credits does the Website Builder Agent use?">Typically over 100 Credits per run. We don't recommend using it unless you have the credits for it.</Accordion>
</AccordionGroup>
