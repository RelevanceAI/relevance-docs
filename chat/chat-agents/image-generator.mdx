title: "Image Generator"

---

The Image Generator Agent is a pre-built Agent that can generate images for you in Chat. You can use it independently in Single-agent mode, or with your other Agents, as well as other Chat Agents, in Multi-agent mode.

<AccordionGroup>
    <Accordion title="How do I use the Image Generator Agent?">You can use the Image Generator Agent by @ mentioning it in Chat.</Accordion>
    <Accordion title="Can I use the Image Generator Agent with my other Agents?">Yes, you can use the Image Generator Agent with your other Agents in Multi-agent mode.</Accordion> 
    <Accordion title="Can I use the Image Generator Agent with other Chat Agents?">Yes, you can use the Image Generator Agent with other Chat Agents in Multi-agent mode.</Accordion>
    <Accordion title="Can I use the Image Generator Agent in Single-agent mode?">Yes, you can use the Image Generator Agent in Single-agent mode.</Accordion>
    <Accordion title="How many credits does the Image Generator Agent use?">Typically over 10 Credits per run.</Accordion>
</AccordionGroup>
