The Deep Researcher Agent is a pre-built Agent that can conduct comprehensive research for you in Chat. You can use it independently in Single-agent mode, or with your other Agents, as well as other Chat Agents, in Multi-agent mode.

<AccordionGroup>
    <Accordion title="How do I use the Deep Researcher Agent?">You can use the Deep Researcher Agent by @ mentioning it in Chat.</Accordion>
    <Accordion title="Can I use the Deep Researcher Agent with my other Agents?">Yes, you can use the Deep Researcher Agent with your other Agents in Multi-agent mode.</Accordion> 
    <Accordion title="Can I use the Deep Researcher Agent with other Chat Agents?">Yes, you can use the Deep Researcher Agent with other Chat Agents in Multi-agent mode.</Accordion>
    <Accordion title="Can I use the Deep Researcher Agent in Single-agent mode?">Yes, you can use the Deep Researcher Agent in Single-agent mode.</Accordion>
    <Accordion title="How many credits does the Deep Researcher Agent use?">Typically over 100 Credits per run. We don't recommend using it unless you have the credits for it.</Accordion>
</AccordionGroup>
