## Step 2: Configure Your Assistant

<Warning>
**‚ö†Ô∏è CRITICAL SETUP - Complete this BEFORE first use**

You must add your API credentials before using your assistant, otherwise you'll see authentication errors.
</Warning>

### What is an API Key?

Think of an API key as a **secure password** that lets the AI assistant access your account on your behalf. It's like giving the assistant a special key to your workspace so it can create, update, and manage things for you‚Äîwithout needing your actual login password.

**Why you need it:**
- The assistant needs permission to make changes in your account
- API keys are safer than sharing your password
- You can revoke the key anytime if needed

### Configuration Checklist

After cloning your assistant, you'll need to provide:

- ‚úÖ **API Key** - A special access token from your integration (see instructions below)
- ‚úÖ **Site/Account ID** - Your specific workspace or site identifier
- ‚úÖ **Documentation** - Pre-loaded (leave as-is, don't modify)

### Where to Add These

1. After cloning, a setup popup should appear automatically ‚Üí Fill in the values
2. If the popup doesn't appear: 
   - Go to the **Variables** tab at the top of your assistant
   - Fill in the required fields
   - Click **Save**

![Variables tab with configuration fields](/images/assistant-variables-tab.png)

## How to Find Your API Key

<Tabs>
  <Tab title="Webflow">
    ### Webflow API Key
    
    **Finding Your API Key:**
    1. Log into your Webflow account
    2. Click your **profile icon** (top-right corner)
    3. Go to **Account Settings**
    4. Navigate to the **Integrations** tab
    5. Scroll to **API Access** section
    6. Click **Generate API Token** (or use existing token)
    7. **Copy the token** - you won't be able to see it again!
    8. Paste into the `webflow_api_key` field in Variables tab
    
    <Warning>
    Copy your API token immediately! Webflow only shows it once for security.
    </Warning>
    
    ![Webflow API key location](/images/webflow-api-key-location.png)
    
    ### Webflow Site ID
    
    **Finding Your Site ID:**
    1. In Webflow, open the site you want to work with
    2. Go to **Project Settings** (gear icon)
    3. Click **General** tab
    4. Find your **Site ID** (long string of letters/numbers)
    5. Copy and paste into `site_id` field in Variables tab
    
    **Alternative:** Your Site ID is also visible in your browser URL when viewing your site
    
    ![Webflow Site ID location](/images/webflow-site-id-location.png)
  </Tab>
  
  <Tab title="Notion">
    ### Notion Integration Token
    
    **Creating Your Integration:**
    1. Go to [www.notion.so/my-integrations](https://www.notion.so/my-integrations)
    2. Click **+ New Integration**
    3. Give it a name (e.g., "Relevance AI Assistant")
    4. Select the workspace you want to connect
    5. Click **Submit**
    6. Copy the **Internal Integration Token** (starts with `secret_`)
    7. Paste into `notion_api_key` field in Variables tab
    
    <Warning>
    **Important Extra Step:** You must share pages/databases with your integration!
    
    - Open the page/database in Notion
    - Click **Share** (top-right)
    - Invite your integration by name
    - Without this step, the assistant can't access your content
    </Warning>
    
    ![Notion integration creation](/images/notion-create-integration.png)
  </Tab>
  
  <Tab title="HubSpot">
    ### HubSpot API Key
    
    **Finding Your API Key:**
    1. Log into your HubSpot account
    2. Click the **Settings** icon (gear, top-right)
    3. In the left sidebar, navigate to **Integrations** ‚Üí **API Key**
    4. Click **Create API Key** (or view existing key)
    5. Click **Show** to reveal your key
    6. Copy and paste into `hubspot_api_key` field in Variables tab
    
    <Note>
    Some HubSpot features require Private App access tokens instead. If your standard API key doesn't work, you may need to create a Private App in Settings ‚Üí Integrations ‚Üí Private Apps.
    </Note>
    
    ![HubSpot API key settings](/images/hubspot-api-key-location.png)
  </Tab>
  
  <Tab title="Salesforce">
    ### Salesforce Credentials
    
    Salesforce requires multiple pieces of information:
    
    **1. Security Token:**
    - Go to Salesforce Setup
    - Navigate to **Personal Information** ‚Üí **Reset My Security Token**
    - Check your email for the token
    
    **2. Username:** Your Salesforce login email
    
    **3. Password:** Your Salesforce login password
    
    **4. Domain:** Your instance URL (e.g., `yourcompany.my.salesforce.com`)
    
    Add all of these to the respective fields in the Variables tab.
    
    ![Salesforce security token reset](/images/salesforce-security-token.png)
  </Tab>
  
  <Tab title="Gmail / Google Workspace">
    ### OAuth Connection (Easy!)
    
    Gmail and Google Workspace assistants use **OAuth**, which means you don't need to manually find API keys!
    
    **Setup Steps:**
    1. After cloning, click the **Add Account** or **Connect** button
    2. Sign in with your Google account
    3. Grant the requested permissions
    4. Done! ‚úÖ
    
    The connection is secure and you can revoke access anytime from your Google Account settings.
    
    ![Google OAuth connection flow](/images/google-oauth-connect.png)
  </Tab>
</Tabs>

## General Tips for Finding API Keys

**Common locations across platforms:**
- Settings ‚Üí Integrations ‚Üí API
- Settings ‚Üí Developer ‚Üí API Keys  
- Settings ‚Üí Security ‚Üí API Access
- Account Settings ‚Üí API & Integrations
- Developer Portal ‚Üí API Credentials

**What to look for:**
- Usually called "API Key", "API Token", "Access Token", or "Secret Key"
- Long string of random letters, numbers, and symbols
- Often starts with a prefix (e.g., `sk_`, `secret_`, `Bearer `)

**Security best practices:**
- ‚úÖ Never share your API key publicly
- ‚úÖ Copy it immediately - some platforms only show it once
- ‚úÖ Store it securely (Relevance AI encrypts it automatically)
- ‚úÖ Revoke old keys if you're no longer using them
- ‚ùå Don't commit API keys to GitHub or public code

**Troubleshooting:**
- Can't find API settings? Look for "Developer", "Advanced", or "Integrations" sections
- Key not working? Check if it has the right permissions/scopes
- Still stuck? Contact the integration's support team for help locating API settings
